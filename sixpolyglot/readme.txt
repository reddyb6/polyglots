A six language polyglot. 
It is a valid code in 6 languages.
C | Shell | Perl | Brainf*** | Befunge | Whitespace


C ignores the meaningless '# define x u' and then prints "hitforum". 

# define x u
main() { printf ("hitforum\n"); }

The shell evals 'echo "hitforum";exit' and exits. 


Perl evals 'echo "hitforum";exit' using sub echo and stops reading at __END__. 

 Brainf*** ignores all characters other than: < > [ ] + - , . 
 
 Befunge reads "# defi", then reverses, comes from right and precedes at ">>>". 
 # defi    v
 >>>>>>>$$$a"muroftih"#[>:#,_@
 
 Whitespace ignores all characters other than:   blank,   tab, newline.
